失序人生 · 完全离线 WEB 版

运行方式
1. 将“WEB离线版文件夹”中的全部内容部署到任意 HTTPS 静态网站根目录。
2. iPhone / iPad 使用 Safari 打开后，通过分享菜单添加到主屏幕。
3. Android 使用 Chrome 打开后，选择安装应用或添加到主屏幕。
4. 首次进入时等待“离线资源准备完成”，之后可以断网游玩。

直接打开 index.html
- 基本游戏和本地存档可以运行。
- 浏览器安全限制会禁止 Service Worker，因此不能安装为 PWA，也不能保证全部资源离线缓存。

存档
- 权威人生档案保存在浏览器 IndexedDB。
- “离线管理”可以导出、导入和清除本机存档。
- 清除浏览器网站数据会同时删除未导出的存档。

语言
- 顶部语言菜单可选择简体中文、日本語或 English。
- 切换语言后页面会自动重新载入，人生、背包、成就和对话存档不会重置。
- 日语与英语译文均已内置，不需要联网翻译。

离线对话
- 不调用任何联网 AI。
- 使用本地角色话术和原有房间权限、属性点扣费、20轮清理规则。

注意
- 离线版不具备防修改能力。
- 所有结局规则和图片均包含在安装包中。
- 请完整保留 assets 目录和所有 JS/CSS 文件。

HTTPS部署
- Cloudflare Pages：上传本文件夹，构建命令留空，输出目录填写“.”。
- Netlify：直接拖入本文件夹，或将站点根目录指向本文件夹。
- Vercel：将项目根目录设为本文件夹，框架选择 Other。
- 详细配置见 DEPLOYMENT.md。
